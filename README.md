# Plain Text Comments

**Plain Text Comments** is a VS Code extension that adds support for toggling `#` comments in plain text files.

## Features

**Plain Text Comments** extends the `plaintext` language configuration to support:

- toggling line comments: `#`
- symbol pairs that **auto-close** and can **surround a selection**:
  - **brackets**
    - `{` `}`
    - `[` `]`
    - `(` `)`

  - **quotes**

    - `"` `"`
    - `'` `'`
